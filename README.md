**Installation**:

DCT depends on

- Python 3
- [PyTorch](https://pytorch.org/) = 1.9.0
- [timm](https://github.com/rwightman/pytorch-image-models)==0.6.11


This repository contains code for evaluation on ImageNet-C, ImageNet-R, and VisDA-2021 with ViT.


**Usage**:

```
bash run.sh
```
